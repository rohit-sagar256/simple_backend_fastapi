# DevOps Lab Roadmap

Project Name: devops-lab

Purpose: This repository is designed to simulate a real-world production
system in order to learn DevOps practices including Docker, CI/CD,
container registries, and Kubernetes deployments.

The application itself will remain intentionally simple. The focus is on
infrastructure, deployment, and production workflows.


# System Architecture

Components included in the system:

-   Backend API (FastAPI)
-   Frontend (React - Vite)
-   Nginx reverse proxy
-   PostgreSQL database
-   Redis
-   Worker (Celery)
-   Scheduler (Celery Beat)
-   Docker containers
-   Docker Compose for local orchestration
-   GitHub Actions for CI/CD
